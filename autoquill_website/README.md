## üöÄ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- macOS (for testing download functionality)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/DevelopedByDev/autoquill_ai.git
cd autoquill_ai/autoquill_website
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Open in browser**
Navigate to `http://localhost:3000` to see the award-winning design in action.
